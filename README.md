# Archivos para GitHub Pages

Esta carpeta contiene todos los archivos necesarios para desplegar la aplicación en GitHub Pages.

## Contenido

- `index.html` - Página principal
- `styles.css` - Estilos
- `app.js` - Lógica de la aplicación

## Despliegue

Sube estos archivos a la raíz de tu repositorio de GitHub y activa GitHub Pages en la configuración del repositorio.
